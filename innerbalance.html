Project overview

A calmer way to approach everyday wellbeing

InnerBalance was developed as a human-computer interaction and UX design project focused on mental wellness. The project explored how thoughtful interface design, supportive language and simple user journeys could make digital wellness tools feel more approachable and less overwhelming.

The interface uses a soft purple visual identity, rounded UI elements, clear typography and uncluttered mobile layouts to create a calm and reassuring experience.

The challenge

Designing support without pressure

Mental-wellness applications need to present sensitive information without making users feel judged, overwhelmed or confused. The challenge was to create a private-feeling, supportive and easy-to-navigate experience that encourages emotional wellbeing.

Emotional Sensitivity

Designing interactions and language that feel supportive rather than clinical or intimidating.

Simple Navigation

Reducing cognitive effort through clear actions, familiar patterns and focused screen layouts.

User Trust

Creating a calm, consistent and private-feeling interface that users can approach with confidence.

Project goals

Wellness design grounded in clarity and care

Create a Calm Experience

Use gentle visual styling and sufficient spacing to reduce visual pressure.

Support Emotional Awareness

Help users reflect on their emotions and wellness patterns.

Improve Accessibility

Use readable typography, clear contrast and understandable interface labels.

Encourage Healthy Habits

Design simple interactions that support consistent wellness routines.

  6. 06Evaluate

    Review the interface, gather feedback and identify opportunities for refinement.

User needs

Understanding User Needs

The design focused on users who may feel stressed, emotionally overwhelmed or unsure how to begin reflecting on their wellbeing.

Clarity

Users need direct language and actions that are easy to understand.

Reassurance

The experience should feel welcoming, private and non-judgemental.

Consistency

Repeated visual patterns help users learn and navigate confidently.

User flow

Simplifying the Wellness Journey

The journey minimises unnecessary decisions. Each stage guides the user toward one focused action while maintaining a clear path back to the main experience.

  1. Welcome or Sign In
  2. Home Dashboard
  3. Emotional Check-In
  4. Wellness Activity or Reflection
  5. Progress Review

Key design decisions

Choices that shaped the experience

Decision 01

Calm Purple Visual Identity

Purple and lavender tones communicate calmness, reflection and emotional support while maintaining a polished digital appearance.

Decision 02

Focused Screen Hierarchy

Each screen prioritises one main action, supported by clear labels, generous spacing and limited competing content.

Decision 03

Familiar Mobile Patterns

Recognisable form fields, buttons, navigation patterns and card layouts reduce the learning effort required from users.

Usability and evaluation

Evaluation and Refinement

The prototype was reviewed to identify navigation issues, inconsistent visual patterns and areas where instructions could be clearer. Feedback informed improvements to hierarchy, button visibility, spacing and interaction consistency.

Clearer Navigation

Improved labels and stronger visual hierarchy make actions easier to locate.

More Consistent Components

Repeated styling helps users understand interactive elements across screens.

Reduced Visual Overload

Focused layouts and additional whitespace support a calmer experience.

Reflection

What I Learned

The InnerBalance project strengthened my understanding of user-centred mobile design and the importance of emotional sensitivity in digital experiences. It also helped me develop practical skills in interface hierarchy, mobile prototyping, design consistency, collaborative project work and presenting UX decisions clearly.
